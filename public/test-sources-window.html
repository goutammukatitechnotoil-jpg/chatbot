🎯 Admin Panel Instructions

  1. Open Admin Panel (requires login)
  2. Navigate to "Appearance" tab
  3. Scroll to "Sources Window" section
  4. Adjust the "Sources Panel Width" slider (200-500px)
  5. View the real-time preview
  6. Click "Save Appearance Settings"
  7. Changes will be applied immediately to the chatbot
