🚀 Quick Start

Integrate the widget into your website with just a few lines of code:

<!-- Add to your HTML --> <script src="https://your-domain.com/chatbot-widget-v2.js"></script> <script> FPTChatbot.init({ tenantId: 'your-tenant-id', apiUrl: 'https://your-api-domain.com', position: 'bottom-right', primaryColor: '#f37021', headerColor: '#1e3a8a', chatbotName: 'FPT Assistant', logoUrl: '/FPTSoftware.png', sliderImages: [ { url: '/slide1.jpg', link: 'https://example.com', alt: 'Slide 1' }, { url: '/slide2.jpg', link: 'https://example.com', alt: 'Slide 2' } ], predefinedSentences: [ 'What services do you offer?', 'How can I contact you?', 'Tell me about pricing' ], buttonIds: { speakToExpert: 'btn_speak_to_expert', continueWithAI: 'btn_continue_with_ai' } }); </script>

💡 Tip: The widget will automatically load configuration from your API if the tenant is set up correctly.

⚙️ Configuration Options

Required Parameters

  • tenantId - Your unique tenant identifier
  • apiUrl - Your API server URL (e.g., https://your-domain.com)

Optional Parameters

  • position - Button position: 'bottom-right' or 'bottom-left' (default: 'bottom-right')
  • primaryColor - Primary theme color (default: '#f37021')
  • headerColor - Header background color (default: '#1e3a8a')
  • chatbotName - Display name (default: 'AI Assistant')
  • logoUrl - Logo image URL (default: '/FPTSoftware.png')
  • zIndex - CSS z-index (default: 9999)

Content Configuration

  • sliderImages - Array of {url, link, alt} objects for welcome screen carousel
  • predefinedSentences - Array of quick question strings
  • buttonIds - Object mapping button actions to tracking IDs

🎨 Design Specifications

Desktop Layout (≥640px)

  • Window size: 1100px × 600px
  • Two-panel layout: Chat panel (flex-1) + Sources panel (300px)
  • Sources panel only visible on screens ≥1024px
  • Rounded corners (12px border-radius)
  • Positioned 24px from edges

Mobile Layout (<640px)

  • Full-screen overlay (100% width/height)
  • No rounded corners
  • Sources panel hidden
  • Vertical button stacking on welcome screen

Welcome Screen

  • Image slider: 300px height, auto-advance every 4s
  • Two CTA buttons: "Speak to Expert" and "Continue with AI"
  • Quick questions section with vertical button list
  • Gray background (#f9fafb) for questions container

Chat Interface

  • Message bubbles: User (primary color) / Bot (gray-100)
  • Avatars: 32px circles (user: gray SVG icon, bot: logo image)
  • Max message width: 70% of container
  • Typing indicator: 3 animated dots with staggered delays
  • Predefined chips: Horizontal scroll, rounded-full pills

📖 API Integration

Required API Endpoints

  • GET /api/config - Retrieve tenant configuration (headers: X-Tenant-ID)
  • POST /api/directline/token - Get Direct Line token
  • POST /api/buttons/click - Track button analytics
  • GET /api/buttons/:buttonId - Get button configuration
  • GET /api/forms/:formId - Fetch form details
  • GET /api/forms/:formId/fields - Get form fields
  • POST /api/lead - Create lead entry
  • GET /api/lead/session/:sessionId - Check existing lead
  • PUT /api/lead/:leadId - Update lead data

Direct Line Integration

  • Connects to Microsoft Bot Framework via Direct Line API
  • Polls for messages every 2 seconds
  • Supports text messages and adaptive cards
  • Extracts sources from channelData/entities

🎯 Migration from V1

If you're using the original chatbot-widget.js, upgrading to V2 is simple:

  1. Replace chatbot-widget.js with chatbot-widget-v2.js
  2. Add new configuration options (sliderImages, predefinedSentences, buttonIds)
  3. Update your tenant configuration in the database
  4. Test the welcome screen and sources panel features

✅ Backward Compatible: V2 works without the new config options - they're all optional!
